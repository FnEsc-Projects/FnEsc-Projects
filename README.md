# Projetos – FnEsc

Projetos desenvolvidos no contexto do laboratório de Física (FnEsc) – UFS.

Este repositório reúne trabalhos voltados à simulação, computação científica, sistemas embarcados e eletrônica, refletindo uma abordagem prática e técnica no desenvolvimento de soluções e experimentos.

---

## Simuladores

- [CD4026 Decade Counter Simulator](https://github.com/FrankSteps/cd4026-decade-counter-simulator)  
  Simulador do decade counter CD4026

- [Digital Counter Simulator](https://github.com/FrankSteps/digital-counter-simulator)  
  Simulação do NE555 e do Johnson counter CD4017

- [Apple Juice Learning Board Simulator](https://github.com/FrankSteps/apple-juice-learning-board-simulator)  
  Simulador de placa didática Apple Juice (aproveitando os conhecimentos adquiridos com os projetos anteriores)
---

## Sistemas embarcados

- [Seed++](https://github.com/FrankSteps/seed-plus-plus)  
  Sistema automatizado de fechadura elétrica projetado com Arduino. O dispositivo é um estudo prático do uso de diferentes paradigmas como POO e POA na construção de um sistema crítico. 

- [Arduino OOP Example](https://github.com/FrankSteps/arduino-oop-example)  
  Experimento de aplicação de programação orientada a objetos no Arduino.

---

## Extras

- [Vector Operations with Haskell](https://github.com/FrankSteps/vector-operations-with-haskell)  
  Implementação de operações vetoriais utilizando Haskell.

Os projetos aqui listados foram desenvolvidos ou estão em desenvolvimento no contexto das atividades realizadas no laboratório, envolvendo estudos, experimentação e implementação prática de conceitos em física e computação tanto teórica como prática.
